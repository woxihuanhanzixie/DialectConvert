# 给文书队友看的技术说明

## 这个项目一句话是什么

《声临其境》是一个把普通话或文字变成方言语音的 AI Demo。它不是单纯做“翻译成方言文字”，而是让用户真正听到一段方言声音。

当前比赛展示版最核心的能力是：

```text
你说一句普通话，或者输入一段文字，系统生成粤语、四川话、闽南语的语音。
```

## 为什么这个项目有意义

很多方言正在慢慢退出年轻人的日常生活。长辈会说，年轻人听得懂一点但不常说，再下一代可能就更少接触了。

传统保护方式通常是录音、建档、写词典，价值很高，但对年轻人来说不够好玩，也不容易传播。

我们的思路是：把方言从“档案里的资料”变成“手机上能听、能玩、能分享的声音体验”。

## 现在 Demo 真正做出来了什么

现在网页能做这些事：

- 输入文字。
- 上传音频或用手机录音。
- 选择目标方言：粤语、四川话、闽南语。
- 调用 CosyVoice v3-flash 生成方言语音。
- 如果实时链路失败，就自动生成兜底音频，保证演示不中断。
- 页面显示耗时、Trace ID、错误/降级原因。
- 页面保留文化百科、RAG 命中、发音提示等展示区，为后续扩展做准备。

## 技术链路用大白话怎么讲

可以这样讲：

第一步，系统先听懂用户说了什么。如果用户上传音频，后端先用 ASR 把语音识别成文字。

第二步，系统把文字整理干净。比如去掉多余空格、处理标点，保证后面的语音合成模型能稳定读出来。

第三步，把整理后的文本交给 CosyVoice。CosyVoice 负责把这段话直接合成为目标方言的声音。

第四步，网页把音频实时播放出来。如果实时生成失败，系统会走非实时兜底，尽量保证现场能听到结果。

## 可以放进 PPT 的技术关键词

- ASR：把语音转成文字。
- LLM：辅助理解和清洗文本，后续可用于更细的语义改写。
- CosyVoice v3-flash：当前主力语音生成模型，负责生成方言音频。
- 声音复刻：有参考音频时，尝试让输出更接近参考说话人的音色。
- 兜底音色：没有参考音频或复刻失败时，用系统音色继续生成，保证 Demo 稳定。
- 三方言支持：粤语、四川话、闽南语。

## PPT 推荐讲法

项目建设方案：

我们针对方言传承弱化、年轻人参与度低、传统保护方式互动性不足的问题，建设一个 AI 方言语音体验平台。用户可以通过文字或语音输入，快速生成目标方言语音，让方言从资料库走向日常表达。

特色与创新点：

项目的创新不是只做文字翻译，而是围绕“能听见的方言”做完整体验。当前 Demo 支持三种方言，结合 ASR、文本清洗、CosyVoice 实时合成和兜底机制，形成可现场演示的语音闭环。

预期成效：

短期形成可展示、可体验的比赛 Demo；中期用于校园方言传播、配音活动和文化展示；长期沉淀方言语料和文化解释，拓展方言树、乡音地图和数字人陪伴。

总结与展望：

声临其境希望用 AI 降低方言表达门槛，让年轻人愿意听、愿意玩、愿意传播家乡话，把濒危方言保护从静态记录变成活态传承。
