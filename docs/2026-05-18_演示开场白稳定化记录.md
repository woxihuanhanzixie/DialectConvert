# 2026-05-18 演示开场白稳定化记录

## 问题

同一句开场白经过 LLM 改写时存在随机性，部分结果会保留普通话表达，例如“你们好”，导致 CosyVoice 即使带方言指令，也可能听起来像普通话在念方言文本。

## 调整

- 将 LLM 改写与情绪分析的 `temperature` 降为 `0`，并设置较低 `top_p`，减少同一句多次请求输出不同。
- 为比赛演示开场白增加固定方言稿兜底：
  - 粤语使用“各位评委老师，大家好，我哋係「声临其境」项目组嘅成员，好荣幸可以参加今次AI应用创新大赛！”
  - 四川话和闽南话也提供固定版本，避免完全依赖模型即时改写。
- 粤语 TTS 指令调整为“请用标准广东话自然表达，不要用普通话发音”，并在演示句命中时追加正式开场白语气提示。
- 增加单元测试，确保演示句命中固定稿，不调用随机改写结果。

## 当前边界

该修复能保证文本改写层稳定，避免“各位老师你们好”这类普通话表达进入 TTS。音色克隆和方言发音仍由 CosyVoice 服务完成，实际音频质量受模型服务状态和参考音频质量影响。
