# Demo1 英文输入链路说明

## 目标

- 支持英文音频输入，并最终输出粤语文本与粤语语音。

## 当前流程

1. 英文音频进入 ASR
2. 执行英文审查
3. 生成 `pivot_text_zh`
4. 基于 `pivot_text_zh` 进行粤语改写
5. 生成粤语语音或克隆音色语音

## 为什么保留 Pivot 中文

- 便于人工审核
- 便于 prompt 调试
- 便于后续扩展其他外语

## 页面展示

- `ASR 原始文本`
- `审查后文本`
- `Pivot 中文`
- `粤语文本`
- `音色克隆状态`

## 当前限制

- 当前英文语言分流优先使用：
  - `FireRedAsr2System + LID`
- 若本地缺少 `FireRedLID` 模型，则自动回退启发式判断。
